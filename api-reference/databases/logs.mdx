---
title: "Logs do Motor"
api: "GET https://api.stackr.lat/v1/databases/{id}/logs"
description: "Recupera as últimas linhas de logs de depuração geradas internamente pelo motor do banco (PostgreSQL/Redis)."
---

### Parâmetros de Rota

<ParamField path="id" type="string" required>
  Identificador do seu banco de dados.
</ParamField>

### Parâmetros de Query

<ParamField query="tail" type="number" default="100">
  Número de linhas de logs a serem retornadas (limite recomendado: `1000`).
</ParamField>

### Resposta de Sucesso (`200 OK`)

```json
{
  "logs": "2026-05-31 19:00:01 UTC [29] LOG:  database system is ready to accept connections\n2026-05-31 19:05:42 UTC [42] LOG:  checkpoint starting: time limit"
}
```
