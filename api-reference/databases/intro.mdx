title: "Introdução aos Bancos de Dados"
description: "A API pública para provisionamento e gestão de bancos de dados PostgreSQL e Redis"
---

A Stackr oferece suporte a bancos de dados de alta performance provisionados sob demanda em nosso cluster. 

A API do banco de dados permite que você crie, liste, reinicie e monitore logs/telemetria dos seus bancos de dados PostgreSQL e Redis de forma totalmente automatizada.

## Mecanismo de Autenticação

Uma vez ativa, a API utiliza o mesmo padrão de Bearer Token (`ApiTokenGuard`) das outras rotas públicas do Stackr:

```http
Authorization: Bearer <seu_token>
```
