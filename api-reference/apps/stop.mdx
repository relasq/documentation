---
title: "Parar Aplicativo"
api: "POST https://api.stackr.lat/v1/apps/{id}/stop"
description: "Desliga com segurança o container da aplicação."
---

### Parâmetros de Rota

<ParamField path="id" type="string" required>
  O identificador UUID v4 da aplicação.
</ParamField>

### Resposta de Sucesso (`200 OK`)

```json
{
  "success": true,
  "status": "STOPPED",
  "message": "Aplicação interrompida com sucesso."
}
```
