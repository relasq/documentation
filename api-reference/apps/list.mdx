---
title: "Listar Aplicações"
api: "GET https://api.stackr.lat/v1/apps"
description: "Retorna a lista completa com todas as aplicações/containers do usuário."
---

### Autenticação

Este endpoint requer autenticação Bearer Token via cabeçalho `Authorization`.

### Resposta de Sucesso (`200 OK`)

A API retornará um array JSON de objetos representando as aplicações:

```json
[
  {
    "id": "a1b2c3d4-e5f6-7a8b-9c0d-1e2f3a4b5c6d",
    "name": "meu-bot-discord",
    "slug": "meu-bot-discord-123",
    "language": "nodejs",
    "status": "RUNNING",
    "memoryLimit": 256,
    "createdAt": "2026-05-30T12:00:00.000Z",
    "updatedAt": "2026-05-31T18:00:00.000Z"
  }
]
```
