---
title: "Estatísticas de Uso"
api: "GET https://api.stackr.lat/v1/apps/{id}/stats"
description: "Retorna a telemetria ao vivo de CPU, RAM e tráfego de rede da aplicação."
---

### Parâmetros de Rota

<ParamField path="id" type="string" required>
  O identificador UUID v4 da aplicação.
</ParamField>

### Resposta de Sucesso (`200 OK`)

```json
{
  "cpuPercentage": 1.45,
  "memoryUsageBytes": 84324128,
  "memoryLimitBytes": 268435456,
  "memoryPercentage": 31.41,
  "networkRxBytes": 124950,
  "networkTxBytes": 98234,
  "online": true
}
```
